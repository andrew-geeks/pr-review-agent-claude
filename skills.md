# General Guidelines
- All code files must follow the best practices.

# Guidelines for Python Files
- All python functions should start with func_ as prefix then function_name. eg: func_calculate

# Guidelines for SQL Files
- All SQL keywords (SELECT, FROM, WHERE, JOIN, INSERT, UPDATE, DELETE, etc.) must be written in UPPERCASE.
- Every table must have a primary key column named `id` and include `created_at` and `updated_at` timestamp columns.
- Avoid using `SELECT *`; always explicitly list the columns you need.
